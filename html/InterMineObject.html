InterMineObject

A base class for all objects instantiated from a ClassDescriptor

This class described the common behaviour for all objects instantiated as representations of classes defined by a ClassDescriptor. It is not intended to be instantiated directly, but inherited from.

_resolve(path) => value
 click to toggle source

Resolve a path represented as a String or as a Path into a value

This is designed to automate access to values in deeply nested objects. So: 

  name = gene._resolve('Gene.organism.name')

Array indices are supported: 

  symbol = gene._resolve('Gene.alleles[3].symbol')


     # File lib/intermine/model.rb, line 227
227:     def _resolve(path)
228:         begin
229:             parts = path.split(/(?:\.|\[|\])/).reject {|x| x.empty?}
230:         rescue NoMethodError
231:             parts = path.elements.map { |x| x.name }
232:         end
233:         root = parts.shift
234:         if !is_a?(root)
235:             raise ArgumentError, "Incompatible path '#{path}': #{self} is not a #{root}"
236:         end
237:         begin
238:             res = parts.inject(self) do |memo, part| 
239:                 part = part.to_i if (memo.is_a?(Array) and part.to_i.to_s == part)
240:                 begin
241:                     new = memo[part]
242:                 rescue TypeError
243:                     raise ArgumentError, "Incompatible path '#{path}' for #{self}, expected an index"
244:                 end
245:                 new
246:             end
247:         rescue IndexError => e
248:             raise ArgumentError, "Incompatible path '#{path}' for #{self}, #{e}"
249:         end
250:         return res
251:     end
